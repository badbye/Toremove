直观上看，收益率在0附近上下波动，样本均值为-0.0161，样本方差3.7408，但不同阶段波动的振幅不同，且振幅大的阶段具有明显的集群性。从相关的背景来看，2007年全国炒股达到高潮，上证指数飙升，于10月17日达到最高6057.43点，之后开始回落，而且08年受到全球金融危机的影响一路暴跌，到2008年10月28日的达到历史最低点1671.4。大涨大跌造成了收益率的强烈震荡。综上所述，初步判断收益率序列具有ARCH效应，可以用ARCH模型拟合。

模型识别： 白噪声检验，平稳性检验

观测序列的基本趋势之后，对数据有了基本的认识，但在建模之前，还要用基本的统计方法对数据进行处理，查看一下隐含的性质，并验证之前序列存在ARCH效应的设想。

先通过自相关函数和偏自相关函数图观测序列是否存在相关性，并对数据做白噪声检验。

par(mfrow = c(2, 1))
acf(rate, main = "ACF of Yield rate")
pacf(rate, main = "PACF of Yield rate")


Box.test(rate)

## 
##  Box-Pierce test
## 
## data:  rate
## X-squared = 11.59, df = 1, p-value = 0.000664

从相关函数图看到，自相关和偏自相关均呈现截尾状态，滞后5介后全为0。另外，Barlett白噪声检验的结果中，\( p \)值趋近于0，即拒绝原价设，认为序列是非白噪声的。因此，序列前后存在一定相关性，可以对其建模拟合。

序列拒绝了白噪声检验并显示一定的相关性，还要对其进行平稳性检验。

# install.packages('tseries')
require(tseries)
adf.test(rate)

## 
##  Augmented Dickey-Fuller Test
## 
## data:  rate
## Dickey-Fuller = -10.68, Lag order = 12, p-value = 0.01
## alternative hypothesis: stationary

根据Augmented Dickey-Fuller单位根检验的结果来看，\( p \)值小于0.01，在5%的显著水平上认为序列不存在单位根，是平稳的。这与序列的集群波动性质不太一致，因为平稳过程的方差的常数，不会随时间的变化而改变。

模型检验

选取和估计好模型之后，应该对模型的残差进行相关性和正态性等性质进行检验。

tsdiag(m1)

上图中，第一张图显示标准化后的残差分布，可以明显的看出前期残差的方差较大，后期很小，再次验证了ARCH效应的存在；第二张图显示残差的自相关函数图，可以认为残差序列不存在自相关；第三张图是对残差进行Ljung–Box检验，与Box–Pierce类似但效果更好，检验残差的分布是否独立，从\( p \)值分布来看不拒绝原价设，认为残差分布是独立的。

综上，残差的方差非齐次与收益率序列不同振幅的波动相符，接下来对残差进行ARCH效应的检验。

ARCH效应检验

为了验证序列是否具有ARCH效应，查看MA(4)模型残差的二阶自相关函数以及偏自相关函数图,以及LM检验的p值分布。

layout(mat = matrix(c(1, 2, 3, 3), byrow = T, 2))
TSA::acf(m1$residuals^2, main = expression("ACF of" ~ residuals^2))
pacf(m1$residuals^2, main = expression("PACF of" ~ residuals^2))
# install.packages('TSA')
require(TSA)
McLeod.Li.test(y = m1$residuals^2, main = "p-values of LM test")

从自相关和偏自相关图中看到，二阶序列的自相关一直不收敛，各滞后相关系数远远超出界限，偏自相关性呈现拖尾性质。Li-McLeod检验的p值图中各阶滞后都在5%的水平上显著，证明收益率序列存在ARCH效应。

ARIMA-GARCH建模

由于收益率序列MA(4)的残差存在ARCH效应，故考虑建立ARMA-ARCH模型。即： \[ \begin{aligned} r_t &= \sum_{i=1}^{p_1} \phi_i r_{t-i} + e_t - \sum_{j=1}^{q_1} \theta_j e_{t-j} \\ e_t &= v_t\sqrt{h_t} \\ h_t &= \alpha_0 + \sum_{i=1}^{p_2}e_{t-i} + \sum_{j=1}^{q_2}h_{t-i|t-i-1} \end{aligned} \]

其中，\( v_t \thicksim N(0, \sigma^2) \)式1是收益率的\( ARMA(p_1, q_1) \)均值模型，式2是均值模型中的误差的方差结构，式3是方差的\( GARCH(p_2, q_2) \)模型形式。

模型定阶和估计

ARMA-GARCH模型的参数是一起估计的，均值模型和条件方差模型的参数会相互影响，与先估计ARMA模型再对残差进行GARCH建模的两步法不同。而两个模型的阶数又很难一起确定，只能不断尝试，在前文MA(4)均值模型的基础上建立最简单的GARCH(1, 1)模型：

所以，得到的最终模型是： \[ \begin{aligned} r_t &= e_t - 0.0469e_{t-j} \\ e_t &= v_t\sqrt{h_t} \\ h_t &= 0.0222 + 0.0681e_{t-i} + 0.9261h_{t-1} \end{aligned} \]
